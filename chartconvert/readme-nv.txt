Reading NV charts 2013/10/15

!!!! consider the NVdigital license restrictions !!!!

1. unpack this zip archive to the chartconvert directory
2. Install OpenCPN + the nv plugin
3. be sure tiler-tools are available
4. call the chartkonverter (replace the pathes by your installations):
-a - path to tiler_tools
-n - path to opencpn
d:\Users\Andreas\Documents\projekte\avnav\chartconvert>read_charts.py -a \Users\Andreas\Documents\projekte-nosync\tiler_tools -n "c:\Program Files (x86)\OpenCPN" \Users\Andreas\Karten-nv-test \Users\Andreas\Documents\Karten\nv\satz2-2012-andreas-win

Be sure that the NV charts can be used inb OpenCPN (try first)....
