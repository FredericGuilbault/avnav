Adapting Avnav

You can tune/adapt AvNav to your needs in various ways.

There are the following options:

  1. adapting the server config file (avnav_server.xml) - not on android
  2. adapting the look and feel
  3. plugins

Adapting the Look and Feel

You can adapt the look and feel in AvNav using various options:

  1. Adapting the value displays (widgets) on the navigation page and on the dashboard pages: adapting the layout.
  2. Creating new display elements using java script. This is described at user defined java script.
  3. Adapting the look and feel via css. As the app itself is browser based, you can adapt the look and feel via css stylesheets. This is described at adapting via css
  4. You can include some user defined HTML pages - both internally stored pages and external urls. This is described at user apps
