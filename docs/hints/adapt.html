Anpassung von Avnav

Avnav kann weitgehend an die Bedürfnisse des Nutzer angepasst werden.

Zur Anpassung gibt es folgende Möglichkeiten:

  1. Anpassung der Serverkonfiguration (avnav_server.xml) - nicht unter Android
  2. Anpassung des Aussehens
  3. Plugins

Anpassung des Aussehens

In AvNav kann man das Aussehen auf verschiedene Art und Weise anpassen:

  1. Anpassung der Anzeigen auf der Navigationsseite und den Dashboard-Seiten (Widgets): Layout-Anpassung.
  2. Einführung neuer Anzeige-Elemente mittels JavaScript. Das ist unter nutzerspezifisches Java Script beschrieben.
  3. Anpassung des Aussehens per Css. Da die App selbst in einem Browser läuft, kann man das Aussehen weitgehend über die Anpassung von Stylesheets ändern. Das ist unter Anpassung per CSS beschrieben
  4. Es können verschiedene separate HTM-Seiten eingebunden werden - das können innerhalb von avnav abgelegte Seiten oder externe URLs sein. Das ist unter User Apps beschrieben.
